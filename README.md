##üí° IntroductionIn C programming, it is often necessary to handle functions that can accept an indefinite number of input parameters. The most famous example of this is `printf`. This repository explores the mechanics behind this feature, offering a clean, modular codebase to learn and experiment with **Variadic Functions**.

Whether you are a student learning C or a developer looking to implement flexible interfaces (like custom loggers or builders), this project serves as a practical foundation.

##üß† What are Variadic Arguments?**Variadic functions** are functions that can take a variable number of arguments. In C, this is achieved using the `<stdarg.h>` header.

A variadic function consists of at least one fixed argument followed by an ellipsis (`...`).
**Example Signature:**

```c
int sum(int count, ...);

```

The `count` (or any other fixed argument) usually provides the necessary context to determine how many additional arguments follow or what their types are (e.g., the format string in `printf`).

##üóù Key Concepts & MacrosThis repository utilizes the standard macros defined in `<stdarg.h>`:

| Macro          | Description                                                                                         |
| -------------- | --------------------------------------------------------------------------------------------------- |
| **`va_list`**  | A data type used to represent the arguments being retrieved.                                        |
| **`va_start`** | Initializes the `va_list` variable. Must be called before accessing arguments.                      |
| **`va_arg`**   | Retrieves the next argument in the parameter list. You must specify the expected **type**.          |
| **`va_end`**   | Cleans up the `va_list`. Must be called before the function returns.                                |
| **`va_copy`**  | (Optional) Makes a copy of the variadic arguments (useful for passing the list to other functions). |

###Common PatternA typical implementation follows this lifecycle:

1. **Declare** a `va_list` variable.
2. **Initialize** it with `va_start` using the last fixed argument.
3. **Loop** through the arguments using `va_arg`.
4. **Clean up** with `va_end`.

##üõ† Implementation Details###Example LogicWhile the specific logic depends on the files in `source-codes/`, a standard integer summer function typically looks like this:

```c
#include <stdarg.h>
#include <stdio.h>

int add_numbers(int count, ...) {
    va_list args;
    int sum = 0;

    // Initialize the argument list
    va_start(args, count);

    // Loop through all arguments
    for (int i = 0; i < count; i++) {
        // Retrieve the next argument as an integer
        sum += va_arg(args, int);
    }

    // Clean up
    va_end(args);

    return sum;
}

```
